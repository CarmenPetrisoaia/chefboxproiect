2. Tema: Proiect final HTML, CSS si Java Script - Site-ul ChefBox

3. Descriere: ChefBox este un Pick-up truck cu mancare pentru pranz situat in zona Victoriei - Strada Buzesti, care ofera zilnic, la alegere, trei meniuri diferite.
In functie de preferinte, clientii pot alege intre feluri de mancare consistente, usoare sau vegeriene.
Pe prima pagina, pe langa bara de navigatie si footer, site-ul contine o scurta introducere, shortcut-uri catre paginile celor trei meniuri si pareri ale clientilor.
Urmatoarea pagina contine meniul zilei. Desigur aceasta pagina ar trebui actualizata in fiecare zi pentru a mentine actualitatea informatiilor.
Galeria Foto cuprinte imagini cu titlu de prezentare pentru cele 3 tipuri de meniuri, atat in format thumbnail cat si sub forma de modal (ambele optimizate).
Pagina de Noutati contine articole si include un clip video si un tabel cu informatii nutritionale.
Ultima pagina a site-ului cuprinde datele de contact pentru ChefBox, o harta care indica pozitia Pick-up Truck-ului si un formular de contact.
